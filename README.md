# 📚 圖書借閱者分類與推薦系統

這是一個基於 Node.js 與 SQLite 打造的全端 Web 應用程式。本系統不僅具備基礎的圖書館借閱管理功能，更結合了**機器學習（Machine Learning）**中的「決策樹（Decision Tree）」演算法，透過分析借閱歷史特徵，將讀者自動分類並提供個人化推薦書單。

---

## ✨ 核心功能 (Features)

### 1. 📖 書籍瀏覽與借閱追蹤
* **書籍列表：** 支援下拉選單，可依據「小說、科普、商業、歷史、漫畫」等類型快速篩選書籍。
* **借閱操作：** 提供直覺的表單，讓使用者輕鬆建立借閱紀錄，並即時寫入後端 SQLite 資料庫。

### 2. 🧠 決策樹讀者分類 (專家系統)
系統透過 SQL 聚合查詢，動態計算每位讀者的兩大核心特徵：
* **總借閱次數** (Total Borrows)
* **單一類型佔比** (Top Genre Ratio - 偏好集中度)

依據特徵數值，系統會運用多層次 `if-else` 決策樹邏輯，將讀者精準分類為四種原型：
* 🔴 **專注型重度讀者：** 借閱量大且極度偏好特定類型。
* 🟣 **博覽型重度讀者：** 借閱量大且涉獵類型廣泛。
* 🟠 **專注型輕度讀者：** 借閱量少但專一。
* 🟢 **探索型輕度讀者：** 借閱量少且類型分散。

### 3. 🎯 個人化書單推薦系統
根據決策樹的分類結果，系統會動態調整推薦策略：
* **專注型讀者：** 推薦其最愛類型中「尚未閱讀過」的熱門/入門書籍。
* **博覽/探索型讀者：** 採用跨類型推薦，隨機挑選各領域的精選好書。

### 4. 🚀 🏆 [進階亮點] 資訊增益 (Information Gain) 最佳化演算
打破傳統「寫死門檻值」的專家系統限制。本專案額外實作了 **C4.5 決策樹演算法**的核心邏輯（詳見 `calcIG.js`）：
* 透過程式自動讀取所有借閱歷史，計算各特徵在不同切分點下的**熵 (Entropy)**。
* 透過比較**資訊增益 (Information Gain)**，讓「機器自己找出」最具鑑別度的最佳切分門檻，徹底落實 Data-Driven 的精神。

## 🛠️ 技術堆疊 (Tech Stack)

* **前端 (Frontend):** HTML5, CSS3, Vanilla JavaScript (單頁應用架構，無需打包工具)
* **後端 (Backend):** Node.js, Express.js
* **資料庫 (Database):** SQLite3 (`better-sqlite3` 套件)
